### Open Port for the TCP/IP Service
---

1. Start "Sql Server Configuration Manager(SQL Server 組態管理員)"
2. 設定通訊埠 : SQL Server 網路組態 > SQLEXPRESS 的通訊協定 > TCP/IP > 內容 > IP 位址 > IPAll > TCP 通訊埠 > 1433
3. SQL Server 服務重啟
