### 安裝
---

* 自 github 安裝 by http protocol

```
# 放置在 ckan 預設放置 plugin 路徑
cd /usr/lib/ckan/default/src/ckan/ckanext

# 取得原 plugin 模組的 http url
$ git clone git@github.com:ckan/ckanext-scheming.git
```
